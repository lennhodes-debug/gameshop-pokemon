## Process

1. Download image from trusted source (PriceCharting, Amazon, eBay)
2. Verify correct product variant
3. Convert to WebP 500x500 quality 85
4. Place in public/images/products/
5. Update src/data/products.json with image path
6. Run npm run build to validate
