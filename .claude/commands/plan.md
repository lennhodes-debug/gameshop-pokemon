Maak een gedetailleerd implementatieplan:

## Stappen

1. **Scope bepalen** — Wat moet er bereikt worden?
   - Gebruik de `planner` subagent voor complexe taken
   - Gebruik de `researcher` subagent om de huidige code te begrijpen

2. **Plan opstellen** met:
   - Concrete stappen op bestandsniveau
   - Agent-toewijzing per stap
   - Afhankelijkheden en volgorde
   - Parallel vs sequentieel
   - File lock planning (wie schrijft waar)
   - Risico's en mitigaties
   - Definition of done

3. **Team kiezen** — Welke team formatie past?
   - Feature Team: nieuwe features bouwen
   - Content Team: teksten, afbeeldingen, SEO
   - Visual Team: animaties, design verbeteringen
   - Quality Team: audits, bug fixes
   - Analyse Team: volledige site-analyse
   - Custom: mix van agents voor specifieke taken

4. **Valideer het plan** — Is het:
   - Uitvoerbaar in de huidige sessie?
   - Consistent met CLAUDE.md conventies?
   - Zonder conflicterende file writes?
