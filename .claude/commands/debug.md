## Stappen

1. **Reproduceer** — Begrijp het exacte probleem
   - Wat is het verwachte gedrag?
   - Wat is het werkelijke gedrag?

2. **Isoleer** — Gebruik de `researcher` subagent om de oorzaak te vinden
   - Traceer de dataflow
   - Identificeer relevante bestanden en regelnummers

3. **Hypothese** — Formuleer max 3 mogelijke oorzaken
   - Rangschik op waarschijnlijkheid

4. **Verifieer** — Test elke hypothese
   - Lees de relevante code
   - Bevestig de root cause

5. **Fix** — Implementeer de minimale oplossing
   - Lees het bestand VOORDAT je wijzigt

6. **Valideer** — `npm run build` om te bevestigen

7. **Commit + push** met Nederlands commit message
