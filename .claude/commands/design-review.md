Visuele en UX review van een component of pagina:

## Stappen

1. **Lees** — Volledige broncode van het component/pagina

2. **Analyseer** op:
   - **Design systeem** — emerald/teal CTAs, navy dark, glassmorphism
   - **Responsiviteit** — Mobile-first, sm/md/lg/xl breakpoints
   - **Animaties** — Framer Motion, niet te druk, prefers-reduced-motion
   - **Per-game thema** — `getGameTheme()` correct gebruikt
   - **Toegankelijkheid** — aria-labels, focus-visible, contrast
   - **UX flow** — Intuïtief, duidelijke CTAs, error states
   - **Typography** — Plus Jakarta Sans, correcte gewichten

3. **Verbeter** — Top 3 verbeterpunten implementeren
   - Lees bestanden VOORDAT je wijzigt
   - `npm run build` na elke wijziging

4. **Build + commit + push**
